Owner can delegate ownership of household to another member

  1. Create a household as user A

  2. Send an invitation to user B’s email address

  3. Logout from user A

  4. Login as user B

  5. Accept invitation from User A to household.

  6. Login as user A again

  7. Click into household

  8. Click “Delete/Delegate/Leave” on the dropdown menu

  9. Select user B as the new owner

  10. Click “Delegate”

After the owner (user A) clicks “Delegate”, the modal will close, showing a green alert and the new owner switched to user B. They will no longer have the option of delegating/deleting the household.
